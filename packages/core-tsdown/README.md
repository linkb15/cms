# Linkbcms Core

This is the core package for LinkbCMS. It contains the core functionality and the shared types.

## Setup

Install the dependencies:

```bash
pnpm install
```

## Get started

Start the dev server:

```bash
pnpm dev
```

Build the app for production:

```bash
pnpm build
```

### Using tsdown

```bash
pnpm dev:tsdown
```

```bash
pnpm build:tsdown
```
